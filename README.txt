Тестовое задание на вакансию «Системный аналитик»

Библиотека вуза хочет автоматизировать свои процессы. Директор библиотеки рассказал следующую информацию: «Хочу, чтобы мои библиотекари вели учёт выданных клиентам книг в электронной форме. И чтобы студенты через Интернет могли просматривать свой читательский билет. И чтобы я (директор) видел статистику по всей библиотеке.».

Необходимо:

Выполнить проектирование предметной области системы (сущности и связи между ними). Например, в виде ER-диаграммы.
Составить список требований к системе, не расписывая их.
Составить описание одного из бизнес-процессов на свой выбор.
Расписать одно из требований на свой выбор.
Сделать прототипы одного из интерфейсов на свой выбор.
//В случае "нехватки информации от заказчика", решения принимать на своё усмотрение.
